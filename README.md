## 🧩 About

This is my personal portfolio, a designated space that shows who I am as a developer, what I've built, and how to reach me. Designed to load fast, and leave a clear impression!

Built with **Astro** due to its simplicity by default, deployed on **Vercel**, and connected with a working contact form by **Resend**.

🔗 **[gzelaya.com](https://gzelaya.com)**

## 💡 Why Astro?

Astro renders **zero JavaScript by default.** Only the components that need interactivity get special JavaScript code, everything else is pure static HTML. For a portfolio where content matters more than client-side logic, this means faster load times and better Core Web Vitals with no extra configuration.

## ⚙️ Local Setup

### Prerequisites

- Node.js 22+

### Steps

```bash
# 1. Clone the repo
git clone https://github.com/GabrielAbarca/gzelaya.git
cd gzelaya

# 2. Install dependencies
npm install

# 3. Configure environment variables
cp .env.example .env
# Add your Resend API key & EMAIL

# 4. Start the dev server
npm run dev
```

### Environment Variables

```env
RESEND_API_KEY=your_resend_api_key
```
